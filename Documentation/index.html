Files Structure

Below are all the files included with the template:

  1. css

HTML Structure

Template grid system is based on a 1200px width, and fullscreen layouts are enabled too.

For responsiveness the grid will support a tablet layout and a mobile layout

  • Between 1024px and 768px wide, the tablet layout will be enabled.
  • Below 768px wide, the mobile layout will be enabled.

JavaScript Structure

All the plugins needed in this template have been listed in files sturcture. The main template javascript file is "scripts.js", down below is the structure of the file.

  • Global section -> contains common functions used globally all over the template pages.
  • Header Section -> contains all the functions used in template menu.
  • Home Section -> contains all the functions used in home layouts.
  • Gallery Section -> contains all the functions used in template galleries.
  • Portfolio Section -> contains all the functions used in template portfolio.
  • Pages Section -> contains all the functions used in pages layouts.
  • Blog Section -> contains all the functions used in template Blog.
  • Responsiveness Section -> contains all the functions used in making the template responsive.

Slideshow

To control slideshow animation, open file "scripts.js" in "js" folder and navigate to Home & Gallery section, line 363, 452.

To change between normal, vertical, horizontal navigation, just change the class of the container to "normal-nav, vertical-nav, horizontal-nav".

Horizontal

To control sliding animation, open file "scripts.js" in "js" folder and navigate to Gallery section, line 504, 602.

To change between normal, centered layout, just change the class of the container to "normal, centered".

Albums

To change between grid, masonry layout, just change the class of the container to "grid, masonry".

Both Grid, Masonry layouts supports 2,3,4 columns. Just change the container class to "cols-2, cols-3, cols-4".

Masonry layout support 3 sizes for albums "size-regular, size-tall, size-wide".

Kenburns

To control sliding animation, open file "scripts.js" in "js" folder and navigate to Gallery section, line 807.

Grid & Masonry

To change between grid, masonry layout, just change the class of the container to "grid, masonry".

Both Grid, Masonry layouts support 2,3,4 columns. Just change the container class to "cols-2, cols-3, cols-4".

Both Grid, Masonry layouts support 2 types of images hover effects (With Icons, With Title) . Just change the container class to "style-icons, style-title".

Both Grid, Masonry layouts support 3 types of layouts spacing (High, Low, No Space) . Just change the container class to "high-space, low-space, no-space".

Masonry layout support 2 sizes for images "size-regular, size-wide".
